# Micro-Wolf

Crossplatform toy game engine that recreates some RtCW (IdTech3) features. It was made with GLFW and support modern rendering via OpenGL 4.1.
My goal is to make game engine like GZDoom, but for RtCW.

Features:
- ✅ rendering IdTech3 .bsp
- ✅ collision detection and player movement
- ✅ 3d-sound via miniaudio
